# Legacy Web Archive

This folder preserves older repository material that is not part of the current Flutter mobile release path.

## What Is Here

- `index.html` - the original single-file web habit app
- `GOOGLE_APPS_SCRIPT_V2.md` - Google Apps Script sync contract from the web-era backend
- `IMPLEMENTATION_PLAN.md` - prior implementation notes for the web product
- `docs/` - architecture and UI diagrams used by the web app docs
- `migration/` - Firebase/Firestore migration helper from an earlier backend transition
- `firebase.json`, `firestore.rules`, `firestore.indexes.json` - Firebase hosting and Firestore config from older work
- `version2_meditation/` - archived prototype/experiment code
- `version3_storytelling.html` - standalone storytelling prototype

## Why It Was Archived

These files were kept because they may still be useful for reference, migration context, or design history, but they are not required to build or release the current Android app.

## Important Note

Anything in this folder should be treated as reference material unless it is deliberately restored into the active app workflow.
